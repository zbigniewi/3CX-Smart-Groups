Privacy Policy

3CX Smart Groups

Last updated: 2026-02-06

1. Overview

3CX Smart Groups is a browser extension that enhances the user interface of the 3CX Web Client by providing user grouping, visual customization, and productivity tools.

This extension is a third-party product and is not affiliated with or endorsed by 3CX Ltd.

2. Data Collection

The extension does not collect, transmit, or store any personal or sensitive user data.

  • No personal identifiers are collected
  • No call data, messages, or credentials are accessed
  • No analytics, tracking, or profiling is performed

3. Local Data Storage

All configuration data is stored locally in the user’s browser using chrome.storage.local.

  • User-defined groups and visual settings
  • Color configurations and UI preferences
  • Custom markers and status icon settings

This data never leaves the user’s device and is not shared with any third parties.

4. Host Permissions and Website Access

The extension requires access to the 3CX Web Client in order to apply its user interface enhancements.

  • Website access is never granted automatically
  • The user must manually enter their 3CX PBX web client URL
  • Permissions are requested only for the specific HTTPS origin provided by the user

The extension does not access websites outside of user-approved domains, does not monitor browsing activity, and does not interact with unrelated websites.

5. Content Script Behavior

Content scripts are injected only on explicitly approved 3CX Web Client pages and operate solely within the 3CX user interface.

  • No cookies or authentication tokens are read
  • No keystrokes or form input are intercepted
  • No communication with external servers occurs

6. Third-Party Services

The extension does not use external APIs, analytics services, advertising networks, or tracking technologies.

7. Data Sharing

No user data is shared, sold, or transferred to any third parties.

8. Security

The extension follows Chrome Extension security best practices:

  • No remote code execution
  • No dynamic script loading from external sources
  • No use of eval() or similar mechanisms

9. Changes to This Policy

This Privacy Policy may be updated to reflect changes in functionality or legal requirements. Any updates will be published on this page.

10. Contact

If you have any questions regarding this Privacy Policy or the extension’s behavior, please contact the extension developer via the Chrome Web Store listing.
